基础
安装、注册github

基础配置
用户名 ： git config --global user.name "your name"
邮箱 : git config --global user.email example@xx.com

初始化仓库:git init
仓库分类：工作区-->(add)暂存区-->(commit)提交版本
-a:add
-m:message
status:用来查看版本库的状态
git commit -a -m "message"

版本回退：
git reset --hard HEAD~1
--hard 代表工作区，暂存区和版本库完全一致

git checkout -- .

git 修改

github 提交代码
git remote add origin git@github.com:Jiayupeng/gitlearn.git
git push -u origin master

git remote:查看远程仓库

上传代码：
git push（是在本地master跟远程master建立关联之后，可以直接使用push）

其他人想要使用代码，可以用clone命令：
git clone <path>

如果推送到远程的时候冲突了，需要先拉取远程的代码到本地，合并之后再push
git pull:拉取远程的代码并且合并
